1.  Entrar al proyecto
2.  Instalar dependencias con composer install
3.  Copiar el archivo .env.example y renombrarlo a .env
4.  Generar la key con php artisan key:generate
5.  Configurar la base de datos en el archivo .env
6.  Ejecutar las migraciones con php artisan migrate
7.  Iniciar el servidor con php artisan serve
